Documentation on operations directly related to the Infinite Improbability Drive blockchain.

## What is this section?

This section contains guides for performing operations directly on the blockchain, including:

- **Network Overview**: Learn about Infinite Improbability Drive, the token, and network identity
- **Token Configuration**: Token denominations and metadata for each network
- **Tokenomics**: Understand the token supply, pool allocation, and distribution strategy
- **Network Parameters**: Complete parameter reference for mainnet, testnet, and creative networks
- **Module Accounts**: Technical documentation for ModuleAccount configuration
- **Vesting Accounts**: Technical documentation for vesting account configuration
- **Genesis Operations**: Create gentxs, validate and work with genesis files, obtain genesis from URL
- **On-Chain Transactions**: Create validators, delegate tokens, transfers
- **Governance**: Create proposals, vote, query proposals
- **Chain Operations**: Chain launch, updates, parameters
- **Queries**: Query account information, validators, staking, network

## Difference with Drive

- **Drive**: How to use Drive as a tool to manage nodes and services
- **Blockchain**: Operations directly with the chain (transactions, genesis, governance) and information about the network itself
